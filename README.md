# React Dashboard

### [React Dashboard](https://dashboard-jet-theta-46.vercel.app/)

 React Dashboard is a React based dashboard template that is built using React,React-Router-Dom, Tailwind CSS.

## Features

- Pages
  - Dashboard
  - Tables
  - Charts
  - Calendar
  - Kanban Board

## Installation

- Clone the repository
- Run `npm install`
- Run `npm run dev`
